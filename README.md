# Panel de Control de Inventario de Vehículos

Este proyecto es una aplicación web interactiva desarrollada con **Streamlit** para analizar datos de anuncios de venta de coches en los Estados Unidos.

## Funcionalidades
* **Visualización de Kilometraje:** Un histograma interactivo para ver la distribución del odómetro.
* **Análisis de Precios:** Un gráfico de dispersión para entender la relación entre el precio y el kilometraje.

## Enlace de la aplicación
Puedes ver la aplicación funcionando en vivo aquí:
[https://panel-vehiculos-claudio-cardona.onrender.com](https://panel-vehiculos-claudio-cardona.onrender.com)

## Cómo ejecutar localmente
1. Clonar el repositorio.
2. Instalar dependencias: `pip install -r requirements.txt`.
3. Ejecutar: `streamlit run app.py`.
